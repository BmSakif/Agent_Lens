## What it does

AgentLens is a Chrome extension that gives AI agents the ability to **see, annotate, and visually communicate** with users on any webpage. Instead of text-only responses, your agent can highlight elements, draw annotations, and provide visual context — making interactions faster and more intuitive.

<table>
<tr><td><b>Element Highlighting</b></td><td>AI agent highlights specific elements on any webpage with labels, colors, and descriptions — so you see exactly what it's talking about.</td></tr>
<tr><td><b>Annotation Drawing</b></td><td>Freehand drawing mode for visual annotations. Draw arrows, circles, and markers directly on the page to call out areas of interest.</td></tr>
<tr><td><b>Wireframe Overlay</b></td><td>Toggle wireframe mode to see the structural layout of any page — boxes, spacing, alignment, and component hierarchy at a glance.</td></tr>
<tr><td><b>Layout Analysis</b></td><td>Visual layout inspector that shows spacing, margins, padding, and alignment between elements.</td></tr>
<tr><td><b>Multi-Selection</b></td><td>Select multiple elements at once for batch inspection, comparison, or annotation.</td></tr>
<tr><td><b>Export & Share</b></td><td>Export annotations, screenshots, and analysis results. Share visual feedback with your team or agent.</td></tr>
</table>

## Quick Start

```
1. Visit any webpage (Dribbble, Figma, GitHub, etc.)
2. Look for the floating toolbar on the right side
3. Click the toolbar to open settings
4. Toggle features on/off in settings
5. Use annotation tools to mark up the page
6. Press Escape or click outside to dismiss
```

---

## How it works

AgentLens runs as a content script injected into every webpage you visit. When the page loads:

1. **Detection** — The extension scans the page DOM and identifies interactive elements (buttons, inputs, images, text blocks, containers).

2. **Overlay** — A floating toolbar appears on the right side of the page with access to all tools and settings.

3. **Annotation** — Your AI agent highlights specific elements with colored labels and descriptions. You see exactly what the agent is referring to.

4. **Drawing** — Toggle drawing mode to freehand annotate directly on the page — arrows, circles, callouts.

5. **Analysis** — Wireframe and layout modes reveal the structural skeleton of any page — spacing, alignment, hierarchy.

6. **Settings** — Click the toolbar to open settings. Enter your license key, toggle features, customize highlight colors, and configure the agent's behavior.
